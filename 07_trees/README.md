# Trees

## Key Concepts

- Recursive structure — most solutions are recursive or use a stack/queue
- DFS (preorder, inorder, postorder) vs BFS (level order)
- BST property: left < node < right — enables binary search on trees

## Common Patterns

1. **DFS recursive**: Base case (null) + recursive calls on children
2. **BFS with queue**: Level-order traversal, shortest path in unweighted tree
3. **BST search/insert**: Use ordering property to go left or right
4. **Path problems**: Track path sum/values from root to leaf
