# Binary Search

## Key Concepts

- Halve the search space each step → O(log n)
- Works on sorted arrays, but also on any monotonic condition
- Key decision: which half to eliminate based on mid value

## Common Patterns

1. **Classic search**: Find target in sorted array
2. **Search on answer**: Binary search on the result space (min/max problems)
3. **Rotated/modified arrays**: Find pivot point, then search
