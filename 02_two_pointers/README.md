# Two Pointers

## Key Concepts

- Use two pointers moving toward each other (or in the same direction) to reduce O(n²) to O(n)
- Works best on sorted arrays or when comparing elements from both ends
- Left/right pointers for pair-finding, fast/slow for cycle detection

## Common Patterns

1. **Opposite ends**: Start at both ends, move inward based on a condition
2. **Same direction**: Fast and slow pointer (linked lists, cycle detection)
3. **Sorted input**: Exploit ordering to skip unnecessary comparisons
