# Stack

## Key Concepts

- LIFO structure — great for matching pairs, tracking state, and backtracking
- Monotonic stack: maintain increasing/decreasing order for "next greater/smaller" problems
- Use for anything with nested or paired structure (parentheses, HTML tags)

## Common Patterns

1. **Matching pairs**: Push opening, pop on closing, check match
2. **Monotonic stack**: Find next greater/smaller element in O(n)
3. **State tracking**: Push state, process, pop to restore
