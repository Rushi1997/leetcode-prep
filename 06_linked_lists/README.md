# Linked Lists

## Key Concepts

- Pointer manipulation — draw it out before coding
- Fast/slow pointer for cycle detection and finding midpoints
- Dummy head node simplifies edge cases (empty list, single node)

## Common Patterns

1. **Two pointers**: Fast/slow for midpoint, cycle detection
2. **Reversal**: Reverse in-place with prev/curr/next
3. **Merge**: Combine two sorted lists with a dummy node
4. **Runner technique**: One pointer ahead by k steps
